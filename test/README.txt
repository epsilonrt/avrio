AvrIO/test README file
$Id$
--------------------------------------------------------------------------------

Ce dossier contient les tests unitaires des modules de AvrIO.
Il permet au développeurs AvrIO de valider leur programme.
Si vous n'êtes pas un développeur expérimenté, il vaut mieux vous rendre dans le
dossier avrio/examples...
Vous pouvez toutefois le consulter afin d'utiliser les modules AvrIo qui n'ont
ni exemple, ni documentation. Vous pouvez prendre  contact avec le développeur
du module pour lui poser des questions ou lui demander de créer un exemple
d'utilisation de son module !

La dernière version de la documentation AvrIo peut être consultée sur:
                http://www.epsilonrt.com/avrio/dev
