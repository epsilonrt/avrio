Rétroéclairage d'afficheur LCD
LCD alphanumérique

Description détaillée

Ce module permet de gérer le rétroéclairage d'un afficheur LCD qu'il soit alphanumérique ou graphique.
Le module utilise un timer en mode PWM pour faire varier l'intensité du rétroéclairage.
La macro AVRIO_LCD_BACKLIGHT_ENABLE dans avrio-config.h et dans ce cas, la partie interface matérielle doit être implémentée dans le fichier avrio-board-lcdbacklight.h .
