Description détaillée

WirelessUSB™ est une marque déposée par Cypress Semiconductor Corporation.

Avertissement
Ce module est en version alpha et n'est pas encore fonctionnel. Il est préférable d'utilser le module Firmware iDwaRF-Net pour l'instant. Dépendances:
  • Fonctions de temporisation
  • Bus SPI
  • Machines d'états
  • Pile d'octets FIFO
