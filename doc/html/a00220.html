EEprom
Noyau temps réel AvrX

Description détaillée

AvrX fournit quelques routines d'accès mémoire EEPROM qui contrôlent l'accès au matériel via un sémaphore. Ce sémaphore doit être libre pour valider l'accès à la mémoire.
