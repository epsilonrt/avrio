Liaison série asynchrone

Description détaillée

Documentation des variables

FILE xSerialPort

Descripteur de fichier de l'uart permettant son utilisation avec les fonctions de la libc.

Pour utiliser l'uart comme entrée et sortie par défaut, on mettra au début du main() :

int main(void) {
vSerialInit (96, SERIAL_DEFAULT);
stdout = &xSerialPort;
stdin = &xSerialPort;
...

On pourra dans le cas contraire passer &xSerialPort au fonction fprintf, fscanf :

int choix;
...
fprintf (&xSerialPort, "Choix ? ");
fscanf (&xSerialPort, "%d", &choix);
...
Exemples:
adc/demo_adc.c, adc_sensor/demo_adc_sensor.c, aprs/demo_aprs.c, idwarf/terminal/hub/terminal_hub.c, keyboard/demo_keyboard.c, rtc/demo_rtc.c, et serial/demo_serial.c.
